# Vacation Inspector App with Devise and Bootstrap
Rails 7 with Devise and Bootstrap 5


## Getting Started

### Requirements
- Ruby 3.1.2
- Rails 7

### Users
- Email
- Password
- Role

### Statement Type
- Name

### Statements
- Start date
- End date
- Status
- Statement type

### Install

```
bundle install
rails db:create
rails db:migrate

rails s
```
